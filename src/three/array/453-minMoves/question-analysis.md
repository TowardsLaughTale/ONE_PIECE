### 453. 最小移动次数使数组元素相等

<br>

### 题目

> 给定一个长度为 n 的非空整数数组，找到让数组所有元素相等的最小操作次数。每次操作将会使 n - 1 个元素增加 1。

<br>

### 示例
```
示例 1:

输入:
[1,2,3]

输出:
3

解释:
只需要3次移动（注意每次移动会增加两个元素的值）：

[1,2,3]  =>  [2,3,3]  =>  [3,4,3]  =>  [4,4,4]
```
