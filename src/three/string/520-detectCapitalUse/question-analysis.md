### 520. 检测大写字母

<br>

### 题目

> 给定一个单词，你需要判断单词的大写使用是否正确。<br>
我们定义，在以下情况时，单词的大写用法是正确的：<br>
全部字母都是大写，比如"USA"。<br>
单词中所有字母都不是大写，比如"leetcode"。<br>
如果单词不只含有一个字母，只有首字母大写， 比如 "Google"。<br>
否则，我们定义这个单词没有正确使用大写字母。

>注意: 输入是由大写和小写拉丁字母组成的非空单词。

<br>

### 示例
```
示例 1:

输入: "USA"
输出: True
```

```
示例 2:

输入: "FlaG"
输出: False
```
