# 递推过程
```js
// 构造数组的两两差
(1)3个下降不满足
\
 \
  \   

(2)2个下降不满足
\
 \/

(3)2个下降不满足
\/\

(4)满足
  /
 /
/

(5)满足
  /
\/

(6)1个下降不确定：需要额外判断下降的幅度
/\/ [1,-1,1]  满足 -1+1>=0 

/\
  \/ [1,-2,1] 不满足 -2+1<0
  
   /  
/\/  [1,-1,2] 满足 -1+2>=0

 /\
/  \/ [2,-2,1] 满足 2+(-2)>=0

(7)满足
 /\
/
```

概括规则
- 只要出现大于等于2个下降的就不满足
- 出现1个下降且下降在中间
    1. 如果下降幅度大于后一个的上升幅度，不满足
    2. 反之，满足
