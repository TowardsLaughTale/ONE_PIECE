# 递推过程
```js
[1] // 0次

[1,2] // 1次
    [2,2]

[1,2,4] // 1+4+1-(2+0)=4次
    [2,2,5]
    [3,3,5]
    [4,4,5]
    [5,5,5]

[1,2,4,5] // 4+5+4-(4+1)=8次
    [2,2,5,6]
    [3,3,5,7]
    [4,4,5,8]
    [6,6,6,9]
    [7,7,7,9]
    [8,8,8,9]
    [9,9,9,9]
```

# 执行过程
```
F(n)moves = F(n-1)moves + F(n)max + F(n-1)moves - (F(n-2)moves + F(n-1)max) 
(n是数组的长度, moves时移动次数，max是当前数组的最大值)

                                    F(6)

                    F(5)                            F(4)

            F(4)             F(3)             F(3)        F(2)

        F(3)     F(2)    F(2)   F(1)      F(2)   F(1)

    F(2)  F(1)   
```
